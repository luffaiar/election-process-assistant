# 🗳 Election Process Assistant

## 🎯 Objective
To help users understand election processes in an interactive and simple way.

## 🚀 Features
- 🧠 NLP-based chatbot
- 🎤 Voice input (speech recognition)
- 🌐 Multi-language support (English, Tamil, Hindi)
- 📊 Election timeline visualization
- 📍 Polling station finder (Google Maps)

## 🧠 Approach
The system uses NLP (CountVectorizer + Cosine Similarity) to understand user queries and provide relevant responses.

## 🔧 Technologies Used
- Python
- Streamlit
- Scikit-learn
- Google Translate API
- SpeechRecognition

## 📌 Assumptions
- Focused on Indian election system
- Designed for first-time voters

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   streamlit run app.py
